This directory holds seed artifacts for the local and EC2 Metabase stacks.

- `metabase.db`: H2 Metabase application database used by this stack. If it doesn't exist, Metabase will create it on first run.
- `hub_analytics.sql`: Optional Postgres dump of the `hub` database (or at least the `analytics` schema) that you can import into the local `postgres` service.

These files can be generated by the export script in `scripts/export-from-apiary.ps1` if you have an existing Apiary stack, but this is entirely optional. The Metabase demo stack will run without them.
