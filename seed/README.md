This directory holds seed artifacts for the local and EC2 Metabase stacks.

- `metabase.db`: Copy of the H2 Metabase application database from the Apiary `metabase-data` volume.
- `hub_analytics.sql`: Postgres dump of the `hub` database (or at least the `analytics` schema) from the Apiary `postgres` service.

These files are generated by the export script in `scripts/export-from-apiary.ps1`.
